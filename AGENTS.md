# AGENTS Rules

1. Maintain the `AGENTS.md` and keep these rules current.
2. Maintain `DESIGN_DECISIONS.md` with clear, dated rationale for important changes. 
3. If you reverse any design decisions, delete the documentation for that original design decision.
4. Keep migrations additive: add new migration files instead of editing applied migrations.
5. Create separate manual-action files using the naming convention `MANUAL_ACTIONS_*.md` in `docs/` for required human steps.
